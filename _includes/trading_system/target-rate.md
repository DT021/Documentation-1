The two possible contexts in which a target rate is defined are the open stage and the close stage. The definition in either stage is valid for the particular stage only.

You may define a rate for each limit order in the corresponding definition under each order. In case there is no definition at the order level, then the target rate for the particular stage is used y default.

If the target rate is not defined for the order, nor is it defined for the corresponding stage, then the system will interrupt the execution with an error.

Market orders do not require a target rate. In the case of market orders, the trading system stores the rate of the close of the candle, instead of the value resulting from the corresponding formulas, like it's the case with limit orders.
